# MovieReviewHub

Простий сайт для оглядів фільмів і серіалів.  
Користувач може переглядати фільми, читати оцінки, додавати свої відгуки.

## Функції
- Перегляд списку фільмів  
- Оцінки та коментарі  
- Пошук  
- Особистий профіль

## Для кого
Для користувачів, які хочуть швидко знайти огляд чи рекомендацію.
